# 🎙️ Zofia – Asystentka TTS PL

Polski asystent głosowy powered by Ollama — TTS/STT, lokalne AI, zero API.

## Możliwości

- Czytanie wiadomości na głos (TTS przez espeak/piper)
- Rozmowa z lokalnym modelem AI (Ollama)
- Ciekawostki, newsy, playlisty muzyczne
- W pełni offline, bez zewnętrznych API

## Instalacja

```bash
chmod +x struktura_start_full.py
python struktura_start_full.py
```

Wymaga: `python3`, `ollama`, `espeak` (lub `piper-tts`)

## Uruchomienie

```bash
python struktura_start_full.py
```

Postępuj zgodnie z menu głosowym.
